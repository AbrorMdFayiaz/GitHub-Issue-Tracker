Answer of the 5 Questions are below :


1. var has functional scope.Often risky to use.
const and let both have block scope. const is fixed for primitive values and modifiable for non-primitives.let is modifiable both for primitive values and non-primitives.

2. spread operator useful tool for copying array elements separately , without copying reference (safe method )

3. foreach runs a function on every array element, but doesn't return anything.map runs a function on every array element and returns the modified array.filter scans on all the array items for a desired condition, then returns an array with the matched items.

4. arrow function is basically a shorter format for writing functions.
it's syntax is ---->  (parameter/s)=>() .
() is used when we need 1 expression which is the return value , and no need to use return keyword.
{} is used when we have many expression and return keyword is used for return value as usual.

5. template literals (``) are a way to represent string , alternative of single('') and double quotes("").it has many benefits over the single and double quotes, which are using expressions , html tags etc.
